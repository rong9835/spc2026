08 — AOS Options Reference

AOS 옵션 정리

속성 목록
data-aos 애니메이션 종류
data-aos-delay 딜레이 (ms)
data-aos-duration 지속시간 (ms)
data-aos-easing 이징 함수
data-aos-once 한 번만 실행
data-aos-offset 트리거 오프셋
초기화 옵션
// AOS.init() 예시
AOS.init({
  duration: 600,
  easing: 'ease-out-cubic',
  once: false,
  mirror: true,
  offset: 100,
  delay: 0,
});

10 — Mirror & Once

Mirror / Once 패턴

data-aos-once="true"

한 번만 실행

스크롤해서 한 번 나타난 뒤에는 위로 스크롤해도 사라지지 않습니다. 대부분의 실무 프로젝트에서 사용하는 설정입니다.

AOS.init({ once: true })
data-aos-mirror="true"

역방향도 실행

위로 스크롤하면 요소가 다시 사라집니다. interactive한 쇼케이스 페이지에 적합한 설정.

AOS.init({ mirror: true })
